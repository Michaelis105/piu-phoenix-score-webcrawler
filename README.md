# piu-phoenix-score-webcrawler
Web crawler concatenates all official Pump It Up player scoring per chart and difficulty

# Usage

1. Login to https://piugame.com/ in browser.
2. Open Inspector or Developer Console.
3. Paste the code into the console.
4. Press "Enter".
5. Wait for code to finish. A CSV-like output should render in the console.
6. Right-click the output then click "Copy Object".
7. Paste into desired text editor.
